# 🎓 Student Performance Prediction Using Machine Learning

## 👩‍💻 Author
Malluri Archana

---

## 📌 Project Overview
This project is a Machine Learning-based system that predicts student performance (final marks) using academic factors like study hours, attendance, assignments, previous marks, and participation.

The model helps in identifying weak students early and supports better academic decision-making.

---

## 🎯 Objective
- Predict student final marks using ML
- Analyze factors affecting student performance
- Help teachers identify weak students early

---

## 📊 Dataset Features
- Study Hours
- Attendance
- Assignments Completed
- Previous Marks
- Participation

**Target Variable:**
- Final Marks

---

## 🤖 Machine Learning Model
- Algorithm Used: Linear Regression
- Type: Supervised Learning
- Output: Continuous value prediction (marks)

---

## 🛠️ Tools & Technologies
- Python
- Google Colab
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

---

## 🔄 Workflow
Data Collection → Data Preprocessing → Train-Test Split → Model Training → Prediction → Evaluation
